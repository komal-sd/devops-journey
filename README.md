## Projects

### 🖥️ Server Monitor Dashboard
Automated monitoring solution for Linux servers

**Features:**
- Real-time disk, memory, CPU monitoring
- Service health checks with alerts
- Timestamped logging system
- Configurable thresholds
- Cron automation

**Usage:**
./scripts/server_monitor.sh

### 🔍 Service Health Checker
Automated service monitoring with instant alerts

**Features:**
- Monitors critical services automatically
- Clear status reporting
- Easy to extend with new services

### 💾 Automated Backup System
Scheduled backup solution with retention policy

**Features:**
- Automated daily backups
- Date-based organization
- Log tracking
